🔧 Connect to Google Drive

1

Get a Google Drive API Key (free)

Go to console.cloud.google.com → Select or create a project → APIs & Services → Library → search "Google Drive API" → Enable → then Credentials → + Create Credentials → API Key. Optionally restrict it to "Drive API" for security.

2

Share "Silverworld Screens" folder publicly

Right-click the Silverworld Screens folder in Drive → Share → Anyone with the link → Viewer. The folder ID is the long string in the URL: drive.google.com/drive/folders/[FOLDER-ID]. Do the same for each location folder and its photo/, video/, and optional audio/ subfolders.

3

Host the files online — required for short links (free, 2 min)

Short player URLs only work when the app is served over https://. The easiest way: go to app.netlify.com/drop, drag your signage-app.zip onto the page, and Netlify gives you a free URL like amazing-name-123.netlify.app. Paste that URL into the Hosting URL field below. You can also rename it to something like silverworld-screens.netlify.app in Netlify settings.
Skip this step if you don't need short links — the app works fully offline without it.

4

Enter your credentials below and discover locations

Paste your API key, folder ID, and optional hosting URL, then click Discover Locations. Your screens will appear with counts, fit controls, and ready-to-use player URLs.

⚠ Running from a local file. Short URLs require the app to be hosted online (see step 3 above). All other features — playback, offline sync, audio — work fine locally.
